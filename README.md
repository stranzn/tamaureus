# ![Project Icon](src-tauri/icons/32x32.png) Tamaureus 

## Summary

Tamaureus is a cross-platform desktop music player built with the tauri framework. Use Tamaureus to listen to all your music completely offline. Great for those who are fed up with streaming services.

## Features

- Playlists
- Adding/removing audio files and editing their metadata
- Themes (light/dark/etc)
- Adapative coloration when adding audio files based on covers

### Planed Features

- yt-dlp integration
- Custom themes
- Ambient mode

## How To Run

```bash
# Open a terminal
# Clone and then CD into the directory

git clone https://github.com/stranzn/tamaureus.git

# Navigate to the project directory
# Run the app

npm run tauri dev

```

## Notes 

Tamaureus is still in early production. The user experience is still a work in progress.
